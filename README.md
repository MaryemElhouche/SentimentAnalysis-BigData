## Dataset

Le projet utilise le dataset [Sentiment140](https://www.kaggle.com/datasets/kazanova/sentiment140) disponible sur Kaggle.

Pour l'utiliser :  
1. Téléchargez le fichier `sentimentsData.csv` depuis Kaggle.  
2. Nettoyez-le et préparez-le comme indiqué dans le projet (`sentimentsData_clean.csv`).  
3. Placez-le dans le dossier `mapreduce/input/` pour exécuter le pipeline MapReduce ou les scripts Pig/Spark.
